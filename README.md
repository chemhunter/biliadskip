# biliadskip

自己写的几个油猴脚本，用于净化B站网页。

1、BiliAdSkip.user.js：

  功能：自动抓取视频字幕，识别广告提醒弹幕，调用AI智能识别广告时间段，自动跳过广告部分。
  
  https://github.com/chemhunter/biliadskip/raw/main/js/BiliAdSkip.user.js

2、BiliCleaner.user.js

  功能：隐藏B站主页、动态页面、评论区、直播间等各处广告和充电专属内容，以及其他美化。
  
  https://github.com/chemhunter/biliadskip/raw/main/js/BiliCleaner.user.js

3、BiliCommentIP.user.js

  功能：B站评论区显示IP归属地。
  
  https://github.com/chemhunter/biliadskip/raw/main/js/BiliCommentIP.user.js
